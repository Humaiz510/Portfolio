Password Validator

Using this application you will be able to test whether your password is valid or invalid according to the program instructions. The two variables that define whether a password is valid or invalid are the length and the inclusion of special characters.

If the password's length is less than 8 characters, it will be invalid. If the password does not include the following characters ('@', '&', '$') it will be invalid. The password will also be invalid if it doesn't meet both requirements.
