Password Generator

Using this application you will be able to generate a strong and unique password. There are multiple variables that help in generating a random password everytime the button is clicked.

The program goes through special characters, lowercase and uppercase letters, and numbers to generate a 10 character length password that is unique everytime the generate button is clicked. The program also allows you to copy the generated password with the click of a button.
